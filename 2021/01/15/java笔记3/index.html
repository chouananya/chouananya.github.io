每日复习

1、标识符的命名规则有哪些？（编译错误）

*26个字母大小写，0-9，——或$组成
*数字不可以开头
*不可以使用关键字和保留字，但能包含关键字和保留字
*Java严格区分大小写，长度无限制

2、标识的命名规范有哪些？

包名：xxxyyyzzz
类名、接口名：XxxYyyZzz
变量名、方法名：xxxYyyZzz
常量名：XXX_YYY_ZZZ

3、数据类型

基本数据类型：
数值型：整数类型（byte（1字节）、short（2）、int（4）、long（8字节））
浮点型：float（4字节）、double（8字节）
字符型：char（1字符=2字节）
布尔型：true、false

引用数据类型：类、接口、数组

4、自动类型转换运算规则

结论：当容量小的数据类型的变量于容量大的数据类型的变量做运算时，结果自动提升为容量大的数据类型。
​ byte、char、short–>int–>long–>float–>double
​特别的：当byte、char、short三种类型做运算时，结果为int类型（包括三种类型作同种运算）
说明：容量大小表示数的大小范围

string无论与哪种类型数据做运算都是连接运算：“+”

5、 强制类型转换：自动类型提升运算的逆运算

容量大—>容量小
1、需要使用强转符：()
2、注意点：强制类型转换，可能导致精度损失。

//整型常量，默认类型为int型
//浮点型常量，默认类型为double型

运算符

算术、赋值、比较（关系）、逻辑、位、三元

算术：+/- 正负

+、-、、/ 加减乘除
% 取余运算

//结果的运算与被模数的符号相同
//开发中，经常使用取模符号判断能否被除尽的情况
//（前）++：先自增1，然后再运算
//（后）++：先运算，后自增1
//–同理

赋值：=

+=、-=、=、/=、%=
//支持连续赋值
//不会改变数据类型

比较运算符：

==/|=/</>/<=?>=
结果都是boolean型（true/false）

逻辑运算符：

&/&&（且）|/||（或）
！（非）
^（异或）相同就false 不同就true

说明：

1、操作的都是boolean型运算符
2、&/&&：1、运算结果相同 2、左边为true时，执行右边的运算 3、左边为false时，&执行右边操作 &&不执行后续操作
|/||1、运算结果相同 2、左边为false时，执行右边的运算 3、左边为truee时，|执行右边操作 ||不执行后续操作
开发中使用&&/||

位运算符（2进制运算）有范围限制
左移 3 << 2 = 12 –> 322 = 12
右移 3 >> 1 = 1 —> 3/2 = 1
无符号右移（空缺位都是用0填补） 3>>>1 = 1 —> 3/2 = 1
操作的都是整型变量
面试题：最高效方式的计算2 * 8 ？ 2 << 3 或 8 << 1

&：只有1&1是1，其余都是0
|：只要有一个是1就是1，其余都是0
^：只要一样就是0，不一样就是1
~：取反，各二进制码按补码各位取反

三元运算符

结构：（条件表达式）？表达式1：表达式2
说明：1、表达式boolean类型 2、条件表达式true/false 执行条件表达式1/2
3、表达式1和表达式2要求是一样的。
4、三元运算符可以嵌套使用。

凡是可以用三元运算符的地方可以改写成if-else结构，反之不一定。
两者皆可时优先选择三元运算符
